# ResilienceOps

ResilienceOps is a fullstack SaaS portfolio project for managing operational risks, incidents and mitigation actions.

The project is built to demonstrate practical experience with:

- Angular
- TypeScript
- Tailwind CSS
- .NET 8
- ASP.NET Core Web API
- Entity Framework Core
- Microsoft Azure
- Application Insights
- OpenTelemetry
- GitHub Actions
- Automated testing

## Project status

The project is currently under active development.

## Purpose

The purpose of the project is to build and document a complete fullstack application from repository creation to cloud deployment.

The project also serves as a learning project where architectural decisions, trade-offs and technical implementation details are documented.
