# Project Goals

## Primary goal

Build a complete fullstack SaaS application using Angular, .NET and Microsoft Azure.

## Learning goals

The project should provide practical experience with:

- Angular components
- Angular routing
- Angular dependency injection
- Angular Signals
- RxJS
- Reactive Forms
- TypeScript
- Tailwind CSS
- ASP.NET Core
- REST APIs
- Dependency injection in .NET
- Entity Framework Core
- SQL databases
- Validation
- Error handling
- Logging
- OpenTelemetry
- Application Insights
- Automated testing
- CI/CD
- Azure deployment

## Development principles

- Every technical choice should have a clear reason.
- Features should be implemented in small, understandable steps.
- Each commit should represent one logical change.
- Code should be testable and maintainable.
- Complexity should only be introduced when it solves a real problem.
